📖 Panduan Setup Google Apps Script v2.1

Ikuti langkah berikut untuk menghubungkan web app ini ke Google Sheets kamu:

  1. Buka Google Sheets Laporan Keuangan
  2. Pastikan ada tiga tab bernama: Data Income, Expenses, dan Settings (jika 'Settings' belum ada, sistem Apps Script akan membuatnya otomatis)
  3. Klik menu Extensions → Apps Script
  4. Hapus semua kode yang ada, lalu paste kode Apps Script terbaru di file AppScript.txt
  5. Simpan, lalu klik Deploy → New Deployment
  6. Pilih type: Web App, Execute as: Me, Who has access: Anyone
  7. Klik Deploy, copy URL yang muncul
  8. Paste URL tersebut di kolom "URL Google Apps Script" di atas
